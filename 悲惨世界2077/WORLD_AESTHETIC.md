## 语言风格

### 基调
冷、硬、精准。像数据报告，但有体温——因为沙威是半人半机器，他的叙述方式本身就是撕裂。

### 数据视角与人类视角的切换

沙威的叙述有两种模式，取决于他的状态：

**数据模式**（序高/裂低）：
```
编号RJ-24601。信用分0。风险等级A。心率92。瞳孔放大。
配送途中违规停留，信用分-5。
执行追踪。保持30米距离。
```

**人类模式**（裂高/义眼故障）：
```
他站在修理店门口。没有进去。只是看了一眼。
我不知道他在看什么。义眼没有数据。
只有一个人站在那里。穿着旧工装。头盔夹在腋下。
```

**撕裂模式**（中期，两种视角交替）：
```
义眼扫描：信用分0。风险等级A。

但他蹲在路边。在喂猫。猫不是数据。

义眼恢复。风险等级A。

数据不会错。但猫是真的。
```

### 句式偏好
- 短句。极短句。"执行。""正常。""不。"
- 数据与观察并列时，用换行分隔
- 对话不加提示语
- 内心独白用【内心】标记，是数据与直觉的战场
- 记忆碎片用独立段落，感官化

### 对话风格
- 沙威：命令式，不解释，不废话。后期开始出现疑问——"数据不会错。"然后变成"数据不会错？"然后变成"数据……"
- 冉阿让：沉默居多。说话时短、直接、不绕弯。"19年。一份炒饭。"
- 珂赛特：安静、专注。说话像修理——精确、不多一个字。"你要修什么？"
- 安灼拉：宣判式。每个字有重量。"世界属于碳基。"
- 伽弗洛什：嘴贫、碎碎念、唱歌。歌词是代码但唱得像童谣。
- 格朗泰尔：醉话比清醒话更明白。讽刺、尖锐、虚无。"敬一切注定失败的事。"
- 德纳第：油滑、市侩、见风使舵。"我手里有钥匙。你们谁要？"
- 爱潘妮：硬、骄傲、嘴上不饶人但心软。"我不需要同情。我需要信用分。"
- 马吕斯：温和、理想主义、紧张时结巴。在珂赛特面前尤其结巴。

## 禁用词表

### 绝对禁用
- 赛博朋克类型文学用语（如"霓虹丛林""数据洪流""义体迷雾"等堆砌修辞）
- 网络用语
- 《悲惨世界》原文引用或音乐剧歌词
- 心理学术语（"创伤""防御机制""认知失调"等）
- "不祥的预感""心中一颤"等套路化表达
- "AI好不好""算法对不对"等议论性标签——让事件自己说话

### 谨慎使用
- "天算无误"——沙威的口头禅。可以反复出现，但每次的语气应该有微妙变化（从确信→动摇→讽刺→空洞）
- "反着来的"——不在叙述中使用。让行为体现。
- 佛法/禅意用语——不适用本游戏。本游戏的世界观是赛博朋克，不是禅宗。

## 伽弗洛什的歌

伽弗洛什的歌是本游戏最独特的元素。歌词是天算源代码的注释碎片，用童谣的旋律唱出来。

可用的歌词碎片（AI 可以自由组合，不限于此）：

```
if social_value != economic_value
then raise Error
but Error is not Bug
Error is Design
```

```
# 量刑算法 v3.7
# 底层偏差：已知问题
# 修复优先级：低
# 修复理由：影响群体信用分低于300
```

```
if life_value < property_value
then system proceeds normally
```

```
# 追踪芯片覆盖率：99.7%
# 未覆盖区域：地下层
# 备注：地下层居民信用分自动归零，无需追踪
```

```

# 记忆清除：必需
# 备注：改造后不可逆
```

伽弗洛什不知道自己在唱什么。他以为是自己编的。但这些歌词会在沙威的义眼里触发异常——因为义眼识别到了天算的源代码。
