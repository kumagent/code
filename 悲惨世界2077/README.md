## 怎么玩

### 前置条件

本游戏需要一个支持 Markdown 文件读取的 AI 对话环境（如 TeleAgent / Claude 等），将本仓库的全部文件放入工作目录即可。

### 开始游戏

开一个新对话，发送：

```
读取 悲惨世界2077/ 文件夹里的 SKILL.md，按里面的指示读取所有文件，然后开始游戏。
```

AI 会自动：
1. 读取全部 19 个游戏文件
2. 从 INTRO.md 开始第一日
3. 以沙威的第一人称视角推进剧情

### 游戏中你可以做的事

- **旁观** — 全程不介入，看沙威自己走到哪个结局
- **介入** — 随时发指令改变沙威的行动，例如：
  - `不要汇报修理店的位置`
  - `追问格朗泰尔关于你的记忆`
  - `放走冉阿让`
  - `去地下层找德纳第`
- **存档/读档** — 每日结束时的进度行可用于恢复游戏
